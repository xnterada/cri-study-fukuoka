# macOS向け Linux環境構築手順

## 概要

macOSユーザーにはLinux環境の構築方法は提供しませんが、代替としてmacOSのターミナルを使用してコマンドを実行する方法を説明します。

> ℹ️ **情報**  
> すでにmacOS上でLinux環境を構築している場合は、この手順をスキップしてください。

## ターミナルの起動

以下に従ってターミナルを起動してください。

1. Finderを開きます。
2. 「アプリケーション」フォルダに移動します。
3. 「ユーティリティ」フォルダを開きます。
4. 「ターミナル」アプリケーションをダブルクリックして起動します。

> 💡 **ヒント**  
> macOSの多くのコマンドがLinuxと共通しています。これは、macOSがUnixをベースとしており、LinuxはそのUnixを参考に作られているためです。
